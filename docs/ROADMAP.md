# TaskGenie 개발 로드맵

> **최종 업데이트**: 2025-11-24  
> **목적**: TaskGenie를 경쟁력 있는 생산성 도구로 발전시키기 위한 통합 계획서

---

## 📊 현재 상태 요약

### ✅ 강점 (이미 구현됨)
- **AI 기반 하위 작업 생성** - 핵심 차별화 포인트 ⭐
- **무한 계층 구조** - 복잡한 프로젝트 관리
- **재귀적 진행률 추적** - 가중치 기반 정확한 계산
- **드래그 앤 드롭** - 직관적인 순서 변경
- **자연어 파싱** - "내일 #긴급 보고서" 같은 빠른 입력
- **반응형 디자인** - 모바일/데스크톱 완벽 대응

### ❌ 개선 필요 영역
- 검색/필터 기능 부족
- 뷰 옵션 제한 (리스트만 가능)
- 통계 및 인사이트 부재
- 협업 기능 없음

---

## 🎯 우선순위 1: 핵심 기능 강화 (High Priority)

### 1.1 케밥 메뉴 기능 확장
**현재**: 수정, 삭제, AI 생성, 우선순위, 마감일 설정  
**추가 필요**:
- [ ] **작업 복제** - 동일한 작업 빠르게 생성
- [ ] **하위 작업 추가** - 현재 작업의 자식으로 추가
- [ ] **프로젝트 이동** - 다른 프로젝트로 드래그 없이 이동
- [ ] **라벨/태그 시스템** - 색상 라벨로 분류
- [ ] **첨부파일 지원** - 파일, 링크 첨부
- [ ] **반복 작업** - 매일/매주/매월 자동 생성

### 1.2 빠른 추가 모달 고도화
**현재**: 자연어 파싱, 수동 우선순위/날짜 선택  
**추가 필요**:
- [ ] **AI 대량 생성 버튼** - "이번 주 할 일 AI로 생성" 프롬프트 기반
- [ ] **작업 템플릿** - 자주 쓰는 패턴 저장/불러오기
- [ ] **태그 입력** (#업무, #개인)
- [ ] **하위 작업 미리보기** - 추가하면서 하위 작업도 입력
- [ ] **음성 입력** - Web Speech API 활용

### 1.3 AI 기능 확장
**현재**: 하위 작업 자동 생성  
**추가 필요**:
- [ ] **스마트 우선순위 제안** - 마감일/중요도 분석
- [ ] **작업 자동 분류** - 입력 텍스트 기반 프로젝트 추천
- [ ] **생산성 인사이트** - "오늘 80% 완료율" AI 피드백
- [ ] **소요 시간 예측** - 과거 데이터 기반 추정

---

## 🔍 우선순위 2: 생산성 도구

### 2.1 스마트 리스트 (Smart Lists)
**목적**: 동적 필터로 효율적 관리
- [ ] **오늘** - 오늘 마감인 작업
- [ ] **중요** - 높은 우선순위 작업
- [ ] **예정됨** - 향후 7일 내 마감
- [ ] **완료됨** - 최근 완료 작업
- [ ] **커스텀 필터** - 사용자 정의 조건

### 2.2 프로젝트 관리 강화
- [ ] **프로젝트 검색/필터** - 사이드바 검색 기능
- [ ] **아카이브** - 완료 프로젝트 숨기기
- [ ] **색상/아이콘** - 프로젝트별 커스터마이징
- [ ] **템플릿** - "개인", "팀" 프로젝트 템플릿
- [ ] **정렬 옵션** - 이름/진행률/마감일순

### 2.3 통계 대시보드
- [ ] **주간/월간 통계** - 막대 그래프
- [ ] **생산성 점수** - Streak, 완료율
- [ ] **가장 생산적인 시간대** - 작업 완료 시간 분석
- [ ] **프로젝트별 소요 시간** - 파이 차트
- [ ] **목표 설정** - "이번 주 20개 완료"

## 🎨 우선순위 3: UX 강화

### 3.1 뷰 옵션 추가
- [ ] **캘린더 뷰** - 시간 기반 작업 관리
- [ ] **칸반 보드** - To Do / In Progress / Done
- [ ] **타임라인 뷰** - 프로젝트 일정 시각화

### 3.2 키보드 단축키 확장
**현재**: `Ctrl+K` (빠른 추가), `Enter`, `Esc`  
**추가**:
- [ ] `/` - 프로젝트 검색
- [ ] `n` - 새 작업
- [ ] `d` - 삭제
- [ ] `e` - 수정
- [ ] `1-3` - 우선순위 변경
- [ ] `?` - 단축키 도움말

### 3.3 알림 시스템
- [ ] **브라우저 푸시** - 마감 1시간 전
- [ ] **이메일 알림** - 중요 작업
- [ ] **데스크톱 알림** - PWA 지원
- [ ] **스누즈 기능** - "10분 후 다시 알림"

### 3.4 기타 UX
- [ ] **다크 모드** - 시스템 설정 자동 감지
- [ ] **드래그 개선** - 프로젝트 간 이동, 일괄 선택
- [ ] **애니메이션 폴리싱** - 부드러운 전환
