# TaskGenie 빠른 개선 요약

## 🎯 핵심 발견사항

### ✅ TaskGenie의 강점
1. **AI 기반 하위 작업 생성** - 경쟁사에 없는 독특한 기능
2. **무한 계층 구조** - 복잡한 프로젝트 관리에 최적
3. **가중치 기반 진행률** - 정확한 진행 상황 추적
4. **우선순위 + 마감일** - 이미 구현됨

### ❌ 개선 필요 영역
1. **검색/필터 기능 없음** - 프로젝트 많아지면 불편
2. **빠른 추가 기능 부족** - 작업 캡처가 번거로움
3. **뷰 옵션 제한** - 리스트만 지원 (캘린더/칸반 없음)
4. **통계 부족** - 생산성 추이 파악 어려움
5. **협업 기능 없음** - 개인용으로만 사용 가능

---

## ⚡ 즉시 구현 가능 (1주일)

### 1. 프로젝트 검색 (2-3시간)
**위치**: `Sidebar.jsx`
```jsx
const [searchQuery, setSearchQuery] = useState('');
const filteredProjects = projects.filter(p => 
  p.keyword.toLowerCase().includes(searchQuery.toLowerCase())
);
```

### 2. 완료 항목 숨기기 (1시간)
```jsx
const [hideCompleted, setHideCompleted] = useState(false);
```

### 3. 항목 개수 표시 (30분)
```jsx
<span className="text-xs text-gray-400">{project.items?.length}개</span>
```

### 4. 기본 단축키 (2시간)
- `Ctrl+K`: 빠른 추가
- `Ctrl+F`: 검색
- `Space`: 체크 토글

**총 예상 시간**: 6-7시간

---

## 🚀 우선순위 높은 기능 (2-4주)

### 1. 전역 빠른 추가 (Quick Add) ⭐⭐⭐
**시간**: 3-4시간  
**효과**: 작업 캡처 속도 3배 향상

```jsx
// Ctrl+K로 어디서나 작업 추가
<QuickAddModal>
  <input placeholder="무엇을 해야 하나요? (예: 내일 오후 3시 회의)" />
</QuickAddModal>
```

### 2. 스마트 리스트 ⭐⭐⭐
**시간**: 4-5시간  
**효과**: TickTick의 핵심 기능 구현

- 📅 오늘 (마감일이 오늘)
- 📆 예정 (7일 이내)
- ⭐ 중요 (우선순위 high)
- ✅ 완료됨

### 3. 캘린더 뷰 ⭐⭐⭐
**시간**: 8-10시간  
**효과**: 시간 기반 작업 관리

```bash
npm install react-big-calendar date-fns
```

### 4. 자연어 파싱 강화 ⭐⭐⭐
**시간**: 6-8시간  
**효과**: Todoist 수준의 빠른 입력

```
입력: "내일 오후 3시 회의 준비 #긴급"
→ 마감일: 내일 15:00, 우선순위: high
```
