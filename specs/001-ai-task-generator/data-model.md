# Data Model: AI Task Generator

## Entities

### ToDoList
- **Description**: Represents a collection of tasks generated by the AI for a specific keyword, associated with a user.
- **Attributes**:
    - `id`: Unique identifier (UUID)
    - `user_id`: Foreign key to the User entity (for authentication/authorization)
    - `keyword`: The keyword provided by the user to generate this list (string)
    - `created_at`: Timestamp of creation (datetime)
    - `updated_at`: Timestamp of last update (datetime)

### ToDoItem
- **Description**: Represents a single, actionable task within a ToDoList. Can be nested under another ToDoItem.
- **Attributes**:
    - `id`: Unique identifier (UUID)
    - `todo_list_id`: Foreign key to the ToDoList entity
    - `parent_id`: Foreign key to another ToDoItem, establishing a parent-child relationship (UUID, optional)
    - `description`: The detailed description of the task (string)
    - `is_completed`: Boolean indicating if the task is completed (boolean, default: false)
    - `order`: Integer to maintain the order of items within a list (integer)
    - `created_at`: Timestamp of creation (datetime)
    - `updated_at`: Timestamp of last update (datetime)

### User
- **Description**: Represents a user of the application.
- **Attributes**:
    - `id`: Unique identifier (UUID)
    - `username`: Unique username (string)
    - `hashed_password`: Hashed password for authentication (string)
    - `email`: User's email address (string, optional)
    - `created_at`: Timestamp of creation (datetime)
    - `updated_at`: Timestamp of last update (datetime)
